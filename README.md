# Poultry Management System

A web-based poultry farm management system built using PHP, MySQL, HTML, CSS, and JavaScript.

## Features
- Admin login system
- Customer authentication
- Poultry product display
- Dashboard with statistics
- Profit calculation
- Sales tracking

## Technologies Used
- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- XAMPP Server

## Installation Guide

1. Install XAMPP
2. Copy the project folder into:
   C:\xampp\htdocs
3. Start Apache and MySQL
4. Open phpMyAdmin
5. Import the database file:
   database/poultry_db.sql
6. Open the project in browser:

http://localhost/poultry_project
